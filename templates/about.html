About This Project

In this project, I have created a sentiment analysis model using Amazon review data. Below are the steps taken to build this project:

  1. First of all, I performed EDA on Amazon reviews.
  2. Second, I performed text preprocessing, including removing URLs, punctuation, stopwords, stemming, lemmatization, etc.
  3. Third, I converted the text into vectors using TF-IDF.
  4. Fourth, I trained different models.
  5. After training different models, I selected the best model and tuned it. After tuning, I saved the model using pickle so that it can be used in the web app.
  6. After saving the model, I created a Flask app and used the model for sentiment analysis.
